## Setup

1. Clone this repository:
   ```bash
   git clone https://github.com/Zlkcyber/rango.git
   cd rango
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Setup: edit config.js and input your wallet address
   ```bash
   nano config.js
   ```
   ![CONFIG](assets/image-3.png)
4. Run The Script:
   ```bash
   npm run start
   ```
